Final Project Basic Description: Flappy Bird
Description: 

	For my final project I will be creating a Flappy bird recreating using Pygame. The game is where you control the bird and keep it flying by pressing a key, while it goes down when nothing is being pressed. The player has to avoid the pipes and obstacles and the more pipes they go through, the higher the score. The game ends when the bird crashes.

Tech used:
Pygame in python

Features:
Bird movement
Gravity
Bird jumpts
Moving pipes
Collision with pipes and ground detection
Scoring
	Scenes:
Flappy bird intro scene
Options to start and objective
Main game screen
Bird in middle
Game over screen
Quit or restart option
Final score
